# אקדמיית החדשנות של גלי

אתר רספונסיבי לתוכנית הקיץ, כולל דף מרכזי לתלמידים ומשחק פתיחה בעברית.

## קבצים מרכזיים

- `index.html` - דף הקייטנה המרכזי.
- `styles.css` - עיצוב רספונסיבי לדף הקייטנה.
- `app.js` - כרטיסי חדרים, מסלול ימים וסימון נוכחות מקומי.
- `משחק_פתיח_אקדמיית_החדשנות.html` - משחק פתיחה לאיסוף אותיות ופיצוח הקוד "חדשנות".

## הרצה מקומית

```bash
python3 -m http.server 8767 --bind 127.0.0.1
```

ואז לפתוח:

```text
http://127.0.0.1:8767/
```
